**AoE** 取名自AI on Edge，是一个终端侧AI集成运行时环境(IRE)。AoE 以“稳定性、易用性、安全性”为设计原则，帮助开发者将不同框架的深度学习算法轻松部署到终端高效执行。

## 功能概述
- 针对Android平台，支持推理框架和数据预处理、后处理运行在独立进程，保障主程序稳定性
- 支持模块拓展注入
- 推理框架执行配置化
- 可支持主流终端推理框架
